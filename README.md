# PythonOC

## Projet 2 Books scrapping

Voici un python script pour le projet 2 d'OpenClassrooms

Le fichier partie2.py est la toute dernière mise à jour du projet. Pour cette version du projet, qui consiste à
récupérer l'info des livres de toute une catégorie, il y a juste la création d'une liste des titres des livres d'une
catégorie tenant en compte le nombre de pages. Cette version est aussi écrite avec des fonctions.

Il manque rajouter le code qui parcourt les pages des livres et récupère l'info des ceux-ci (lequel se trouve commenté
de façon prévisionnelles).

## Installation

Python version : 3.9
<création du venv>

Les dépendances sont listées dans le fichier `requirements.txt`
Lancer
```
pip install -r requirements.txt
```
